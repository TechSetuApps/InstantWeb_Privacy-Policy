Privacy Policy No Personal Data Collected 📢 Contains Ads

Overview

InstantWeb is a local HTML file hosting tool by TechSetuApps. We are fully committed to your privacy. This policy explains what data we access and why.

Data We Do NOT Collect

  • No personal information is collected or stored remotely by us.
  • No usage analytics or crash reports are sent to any server owned by us.
  • Files you serve are never uploaded to any cloud or third-party server by us.
  • No account or registration is required.

Advertising (Google AdMob)

InstantWeb displays advertisements via Google AdMob to support free development. A short interstitial ad is shown before each hosting session. A 2-minute grace period applies — re-hosting within 2 minutes does not show another ad.

AdMob may collect and use data to serve relevant ads, including:

  • Device advertising identifier (Advertising ID)
  • IP address and approximate location
  • App interaction data for ad relevance

Google's data collection is governed by Google's Privacy Policy at policies.google.com/privacy. We do not control or access the data collected by AdMob.

Local Storage Only

InstantWeb temporarily copies selected HTML files to your device's local cache solely to serve them over the local HTTP server. This data never leaves your device and is deleted when hosting stops.

Network Access

The app creates a local HTTP server on your device (default port 7090), accessible only by devices on the same local network. No personal data is transmitted to the internet through this server.

App Update Check

On startup, InstantWeb silently checks for updates by fetching a small version file from GitHub. No personal data is sent — only the app version is compared locally. Fails silently if offline.

Permissions Used

  • INTERNET: To run the local HTTP server, load ads, and check for updates.
  • ACCESS_WIFI_STATE / ACCESS_NETWORK_STATE: To detect your hotspot IP address.
  • READ_EXTERNAL_STORAGE / READ_MEDIA: To select HTML files from your device.
  • POST_NOTIFICATIONS: To show the "Hosting Active" notification.
  • FOREGROUND_SERVICE: To keep the server running in the background.

Children's Privacy

InstantWeb is not directed at children under 13. We do not knowingly collect any information from children under 13. If you are under 13, you must not use this app.
