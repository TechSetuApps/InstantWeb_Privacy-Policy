Privacy Policy No Personal Data Contains Ads

Last updated: April 2026

Overview

InstantWeb is a local HTML file hosting tool by TechSetuApps. We are fully committed to your privacy.

Data We Do NOT Collect

  • No personal information is collected or stored remotely by us.
  • No usage analytics or crash reports are sent to any server owned by us.
  • Files you serve are never uploaded to any cloud or third-party server.
  • No account or registration is required.

Advertising (Google AdMob)

InstantWeb displays ads via Google AdMob to support free development. A short interstitial ad appears before each hosting session. A 2-minute grace period applies.

AdMob may collect device advertising ID, IP address, and app interaction data. Governed by Google's Privacy Policy.

Local Storage Only

InstantWeb temporarily copies selected files to your device's local cache to serve them. This data never leaves your device and is cleared when hosting stops.

Network Access

The app creates a local HTTP server on your device (default port 7090), accessible only on the same local network. No personal data is transmitted to the internet.

App Update Check

On startup, InstantWeb fetches a small version file from GitHub. No personal data is sent — only the version number is compared locally.

Permissions Used

  • INTERNET — Run the local HTTP server, load ads, and check for updates.
  • ACCESS_WIFI_STATE / ACCESS_NETWORK_STATE — Detect your hotspot IP address.
  • READ_EXTERNAL_STORAGE / READ_MEDIA — Select files from your device.
  • POST_NOTIFICATIONS — Show the "Hosting Active" persistent notification.
  • FOREGROUND_SERVICE — Keep the server running in the background.
  • WAKE_LOCK — Prevent Android from stopping the server due to battery optimization.

Children's Privacy

InstantWeb is not directed at children under 13. We do not knowingly collect information from children under 13.
